アプリケーション種別はクライアントアプリケーションの基本的なカテゴリーを示します。 web もしくは native が選択可能です。 もしくは、どちらも選ばないとすることもできます。

この設定パラメーターの値はリダイレクト URI の検証手順に影響します。 web が選択された場合は、implicit フローで使用するリダイレクト URI のスキームは https でなければならず、また、ホスト名に localhost を用いてはなりません。 一方、native が選択された場合は、リダイレクト URI のスキームはカスタムスキーム、もしくはホスト名が localhost の場合に限り http、とすることができます。

この設定パラメーターは OpenID Connect Dynamic Client Registration 1.0 の 2. Client Metadata で定義されている application_type に対応します。
